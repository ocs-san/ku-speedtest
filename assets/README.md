# Assets Directory

## Logo File
Please add your Kasetsart University logo file here:
- **File name:** `KU_Logo_PNG.png`
- **Recommended size:** 300px width (height will auto-scale)
- **Format:** PNG with transparent background

The logo will be displayed at the top of the speed test page.

If the logo file is not found, the page will fallback to displaying "KU Speed Test" as text.
